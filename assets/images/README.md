# Images

Put your photographs in this folder.

Suggested files:

- `portrait.jpg` - a portrait for the hero section. A vertical photo with a
  4 by 5 shape looks best. Then set `deceased.portrait` in `js/config.js` to
  `assets/images/portrait.jpg`.
- `tribute.jpg` - an optional second photo for the tribute section. Set
  `tribute.photo` in `js/config.js` to `assets/images/tribute.jpg`.

Tips:

- Use JPG or PNG files.
- Keep each image under about 1 megabyte so the page loads quickly. You can
  shrink a photo for free with your computer's built-in photo tools.
- Always write a short description in the matching `...Alt` setting in
  `config.js` so screen reader users know what the photo shows.

If you do not add any photos, the site shows a tasteful placeholder and still
looks complete.
