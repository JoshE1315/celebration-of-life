# Documentation Guide

These documents walk you through everything, in plain language. Read them in
this order.

1. **SETUP.md** - the big picture and how to preview the site on your computer.
2. **GOOGLE-SHEETS-SETUP.md** - create the spreadsheet and connect the backend.
3. **DEPLOYMENT.md** - put the website online with free GitHub Pages.
4. **CUSTOMIZATION.md** - change the words, colors, photos, and details.
5. **PRIVACY.md** - what is collected and how it is protected.
6. **TESTING.md** - a checklist to confirm everything works.

If you only have a few minutes, start with SETUP.md. You can run the whole site
in a safe demonstration mode before you set up Google or GitHub.
